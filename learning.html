Why Did I Learn React?

So why did I learn React? I wanted to learn React because throughout my job serach I have seen many businesses looking for someone with knowledge in this JavaScript library.

I also wanted to learn React because I don't want to just stop on one framework, library or programming language. I want to know and learn as many as possible and I felt React was one that was very important to know! Especially as I move onto various other things and ways of creating things.

Will I use it in the future?

Using React in the future is something I can't quite decide on! On one hand, I do like how much cleaner and easier it is to create a full web application but on the other I just can't see myself using it on one of my own big projects.

So as much as I like React, I don't think I will use it for myself.

How I found React?

I found React relatively easy to learn. The idea to create a web application or website in such a way that makes everything seem so much cleaner, especially when it comes to the code, makes me pretty happy!
